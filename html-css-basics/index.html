Элементы группировки - нужны для структуризации контента и заключения содержимого в отдельные контейнеры:

    - параграфы, абзацы текста. Использовать чисто для текста, но можно и картинки включать.

    - заголовки. h1 (заголовок 1-го уровня) должен быть на странице ТОЛЬКО 1, остальные - хоть сколько, но, например, не должно быть h2 без h1, h3 без h2 и т.д. Пример: есть 2 цены, 1 из которых - зачёркнута => робот поймёт семантику, в отличие от стилизованной. 2) Если файл CSS не прогрузится, то не отобразится зачёркивание. P.S.
