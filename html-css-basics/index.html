Теги делятся на: 1) Теги содержания - используются для контента:

  1. ... 2) Семантические - нужны для роботов, чтобы понимали, какой контент и где содержится: * Меню:
    * Основная часть страницы:
    * Сайдбар (новостная информация):
    * Секция:
    ...? Классов может быть несколько => перечисляются через пробел: Спецсимволы: *   " " - неразрывный пробел. Нужен, например, для переноса слов по правилам русского языка, или для написания цены в одну строчку (только при хватке места в строке): 5 000 руб. => 5 000 руб. > < & ... Элементы группировки - нужны для структуризации контента и заключения содержимого в отдельные контейнеры:

    - параграфы, абзацы текста. Использовать чисто для текста, но можно и картинки включать.

    - заголовки. h1 (заголовок 1-го уровня) должен быть на странице ТОЛЬКО 1, остальные - хоть сколько, но, например, не должно быть h2 без h1, h3 без h2 и т.д. Примеры тегов для форматирования текста: жирный жирный со смыслом (говорит роботу о важности) курсив важный курсивdasdda выделенный (тоже важный) зачёркнутый - ВСЕГДА ИСПОЛЬЗОВАТЬ, а не зачёркивать в CSS, т.к. 1) Роботам и скринридерам, которые не анализируют CSS, понятна семантика 2) Если файл CSS не прогрузится, то не отобразится зачёркивание. P.S. В таких случая будет использоваться вёрстка с браузера тоже зачёркнутый ПОДчёркнутый текст, размер которого чуть меньше, чем у окружающего апод строкойа анад строкойа авставленныйа Формы:
